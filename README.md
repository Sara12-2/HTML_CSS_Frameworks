# 🎯 Best Practices

- Write semantic HTML.
- Keep CSS reusable.
- Prefer Flexbox for one-dimensional layouts.
- Prefer Grid for two-dimensional layouts.
- Build mobile-first.
- Optimize images.
- Minify CSS.
- Use accessible colors.
- Add alt text.
- Test responsiveness.
